How to use get selector script:
First, add this code snippet right before closing the <body> tag.
<script src="https://justin226.github.io/scripts/getselector.js"></script>
How to use:
To get a element by a class name:
get(".class-name")
To get element by an id:
get("#id") To get all elements of a tag name:
get("tag-name")
And that's it!
